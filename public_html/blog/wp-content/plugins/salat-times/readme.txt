== Description ==

- Install and activate the plugin.

== Installation ==

1. Download the plugin.
2. Go to: Dashboard > Plugins > Add New > Upload.
3. Upload, install and activate the plugin.
4. Done!
5. Use widget: "Daily Salat Times".

== Frequently Asked Questions ==

= How does it work? =
- Widget: "Daily Salat Times"
- Shortcode: [daily_salat_times]
- PHP Code: <?php echo do_shortcode('[daily_salat_times]'); ?>


== Upgrade Notice ==

After upgrading to version 2.0 from version 1.3 or older, click on "Restore Default Settings" button on Salat Times Settings page. Otherwise widget may not work properly.
